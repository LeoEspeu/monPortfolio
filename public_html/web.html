GLPI-OCS: Gestion d'un parc informatique

Ce TP a été réalisé en SI7 ,il avait pour objectif d'installer l'outil de gestion de parc informatique GLPI (Gestion Libre de Parc Informatique) couplé à OCS-INVENTORY (réalise l'inventaire du parc informatique) ,il avait d'abord été demandé de créé un réseau privé interne composé de 3 machines:

  • Serveur Debian (serveur GLPI).

  • Debian graphiqe.

  • Machine Windows 10.

Ainsi que d'un routeur pour assurer l'accés à internet.
GLPI est un outil qui permet de gérer,en fonction du niveau sélectionnés dans l'organisation hiérarchique de lieu de l'entreprise, la remontée d'incident,l'envoi et la gestion de ticket ,l'amortissement du matériel et la partie comptabilité.

GSB

GSB(Galaxy Swis Bourdin ) est un projet de site web permettant de réaliser les tâches administratives pour l'entreprise fictive du même nom ,il a été réalisé avec les technologies HTML5,CSS3,PHP 7,MYSQL,javascript et le framework CSS bootstrap 3.3.7. Ce projet ,réalisés en binome et versionnés sur la plateforme Github, nous servira de support pour l'épreuve E4 du BTS.


Le projet a été réalisés sur un existant fournit par le professeur ,il y avait l'obligation de respecter l'architecture MVC (modèle-vue-contrôleur) de celui-ci ,la partie visiteur étant déja codée notre mission était de codé la partie comptable du sîte ainsi que d'un service Windows pour automatiser la plupart des tâches (projet en c#).
