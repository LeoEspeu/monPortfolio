Site wed de Nolark

Ce TP a pour contexte la construction du site web de l'entreprise fictif "Nolark" qui est un magasin de vente de casque . Il nous a permis d'aborder les notions de base du développement web avec la mise en pratique de langage tel que le HTML5 ,le CSS3,le javascript ,le framework CSS Bootstrap. Il nous a aussi de support pour les premières notions de PHP.
