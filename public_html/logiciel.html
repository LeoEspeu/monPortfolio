Projet Oxford

Le projet Oxford est une réalisation de reconnaissance faciale ,dans un contexte d'authentification pour accéder au locaux d'une entreprise ,la reconnaissance faciale fut l'axe principale du projet,elle a été réalisée grace à une API MICROSOFT.


Trois équipes étaient sur ce projet : une équipe sur la partie PHP pour développer une application web d'administration et de gestion des employés, et deux équipes sur la partie C# : une pour la partie inscription de l'employé (dont j'ai contribué), et la seconde pour la partie correspondance du profil avec un profil stocké dans une base de données MySQL.


L'organisation du projet était la suivante : chaque mardi soir nous avions une réunion en visio conférence pour parler de l'avancée du projet et des difficultés rencontrées, nous utilisions Trello pour que tout le monde puisse suivre l'avancement global du projet, et nous utilisions pour la même raison des kanbans en classe. Chaque étape du projet était définie dans un sprint.

Maison des ligues

Le projet maison des ligues est le second projet en PPE qui est en cours de réalisation dans l'optique de l'épreuve E4 ,le projet a été réalisés en trinome en appliccation avec les méthodes agile (mise en place d'un tableau des tâches sur Trello ,réunion hebdomadaire ) , à la différence du précédent projet nous avons choisi de versionner celui-ci sur la plateforme de travail collaboratif pour les projets de Visual Studio ,Team Foundation ,par les nombreuses outils quelle apporte en agilité de travail.


Le projet est à nouveau basé sur un existant (solution Visual Studio 2013),c'est une application lourde utilisant les technologies c#,Visual Studio 2017,sql/plsql sur Oracle et Entity Framework.
